# IDRC WCAG Reporter

The IDRC WCAG Reporter lets you create WCAG Audit Reports from Markdown files. It is based on [Eleventy WCAG Reporter](https://github.com/hidde/eleventy-wcag-reporter)
by Hidde de Vries.
